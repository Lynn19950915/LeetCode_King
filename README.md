### 專案簡介

宗教機器人是一款 `LINE 自動聊天程式`，它鼓勵用戶們轉傳、分享自己讀到有感的宗教訊息（可以是一段話、一張圖片或一段語音）。

透過問題的提問及訊息的收集，我們希望能捕捉用戶**在線上的日常宗教接觸**狀況。
